To run the project first go to CHATBOT 
$ cd CHAtBOT
$ install npm
$ npm run dev

And go to TEMPGEN
 $ cd..
 $ cd TEMPGEN
 $ npm install
 $ npm run dev

 And now to go the Canva-Clone
  $ cd..
  $ cd canva-clone-react-cesdk-main
  $ npm install
  $ npm start

  Now go to the CONTENTQL
   $ cd..
   $ cd Content QL 3.2
   $ npm install
   $ npm run dev


   Place your API key in the TEMPGEN folder located as ".env.local"
